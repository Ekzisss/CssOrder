# CssOrder

This extension order your css rules.

Order of rules:

1. Positioning
2. Sizing
3. Text
4. Other

![example](https://github.com/Ekzisss/CssOrder/blob/master/src/example.png?raw=true 'example')

## Usage

`F1` > `CssOrder: Format`

> Works only in .css files
